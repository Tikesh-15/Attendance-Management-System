# 🎓 Student Attendance Management System

Ek modern aur secure Desktop Application jo college ke attendance aur records ko manage karti hai.

## 📱 Project Demo
![App Demo](https://github.com/user-attachments/assets/47a4ff4b-033c-4c7d-b349-894354415d8b)## 🚀 Main Features
* **Dashboard:** Real-time stats (Total Students, Staff, Courses).
* **Attendance:** Subject-wise marking with duplicate entry protection.
* **Management:** Full CRUD for Students, Staff, and Courses.
* **Search:** Advanced filters to find records instantly.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **GUI:** Tkinter
* **Database:** SQLite3

## 💻 How to Run
1. Install dependencies: 
`pip install -r requirements.txt`
2. Run the app: `python main.py`
